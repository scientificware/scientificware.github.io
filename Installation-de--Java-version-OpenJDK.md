# Windows 10 :
Télécharger la dernière version : à partir du site [jdk.java.net/](http://jdk.java.net/)

Vérifier la signature avec : `certutil`
Exemple : `certutil -hashfile openjdk-11.0.1_windows-x64_bin.zip SHA256`
